# 项目简介
**Merge Tactics Tool** 是一款针对**皇室战争：合合奇兵**（**Clash Royale: Merge Tactics**）的阵容推荐工具，基于 `C#` 和 `.NET` 开发。
> 目前该工具还在开发中，功能不完全、不稳定，更新不定期

## 使用方法
从 `Release` 处下载压缩包，解压后双击 `MTtool.exe` 执行。
界面左侧用卡牌组成一个表格，同行、列卡牌具有相同羁绊。左键点击卡牌以选中/取消选中，右键点击卡牌以禁用/取消禁用。
界面下方折叠栏提供额外羁绊，可在开局设置。左键点击图标使羁绊个数 +1，最多为 2 个；右键点击以归零。
界面右侧根据表格中选择和禁用的卡牌生成阵容推荐，共 10 个，下方有计算时间提示。

## 其他文件
`data/` 下存放有关卡牌和羁绊的配置文件，`image/` 下存放有关图片。
